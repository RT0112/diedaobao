# K70 网络断连排查 (12:28)

## 现象
- cloudflared tunnel DNS 查询超时
- ngrok DNS 查询超时  
- curl http://google.com 超时 (HTTP 000)
- curl http://8.8.8.8 超时

## 根因
K70 设备网络完全中断（不只是 DNS，TCP 也通不了）

## resolv.conf 配置
nameserver 8.8.8.8
nameserver 1.1.1.1

## 待排查
- K70 WiFi 是否断开
- 路由器是否限制/过滤
- 手机是否飞行模式

## 备选方案
1. 检查 K70 WiFi 状态（用户手动确认）
2. Mac 做隧道转发（cloudflared tunnel --url http://192.168.4.19:3000）
3. 另一台手机连同一 WiFi 直接用局域网 IP
