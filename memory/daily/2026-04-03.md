## 关键路径问题（重要！）

- 每个 agent 有独立 workspace：`/Users/mac/.qclaw/workspace-agent-<ID>/`
- **不要**用共享 workspace 路径（`/Users/mac/.qclaw/workspace/`），那是默认/共享目录
- cron 任务路径必须精确指向目标 agent 的 workspace 目录
