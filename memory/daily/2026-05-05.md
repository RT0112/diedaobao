App 级别的通知重要性是 DEFAULT（不是 HIGH 或 MAX），限制了所有通知的展示能力。

|------|-------------|
| fall_alert_channel | 4 (HIGH) ✅ |

- fullScreenIntent + PRIORITY_HIGH + CATEGORY_CALL + CATEGORY_CALL
- setAutoCancel(true) + setTimeoutAfter(60s)
- 应该能在锁屏显示并弹出全屏 Activity

通知创建了但可能在 MIUI 上被折叠或管理了。通知重要性 DEFAULT 意味着即使 PRIORITY_HIGH，MIUI 也可能只显示在通知列表而不横幅。
