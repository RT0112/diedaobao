## 遇到的问题
编译错误：FallDetectionService.kt 第37行 `mlInferCount`/`mlFallCount` unresolved reference

**根因**：`mlInferCount` 和 `mlFallCount` 定义在 FallDetector 的 companion object 中，通过实例 `_currentInstance?.fallDetector?.mlInferCount` 访问时 Kotlin 编译器无法正确解析（因为 companion object 成员应该通过类名直接访问）。

**修复**：
```kotlin
// 修复前（错误）
"ML: " + (_currentInstance?.fallDetector?.mlInferCount ?: 0) + " 跌倒: " + (_currentInstance?.fallDetector?.mlFallCount ?: 0)

// 修复后（正确）
"ML: " + FallDetector.mlInferCount + " 跌倒: " + FallDetector.mlFallCount
```
