# 任务总结 2026-04-25 12:14

## 目标
修复跌倒检测"一直没到 ML"的 bug

## 根因
`FallDetectorONNX.gravityAlign()` 存在双重 GRAVITY 除法 bug：
- SensorCollector 已将 m/s² → g（÷9.8）
- gravityAlign 又除以 GRAVITY（9.81），导致所有特征值偏小 ~10 倍
- 静止手机 accMag 应为 ~1.0g，实际为 ~0.1g
- ML 标准化后特征完全偏离训练分布，输出永远 ≈ 0

## 修复
1. 移除 gravityAlign 中所有 `/GRAVITY` 除法
2. 修正水平分量投影：旧 `gDotA/(GRAVITY²)*GRAVITY` → 标准 `accVec - gravDir × gDotA`
3. 归一化重力方向后存回 smoothedGravity，避免后续索引不一致

- Finder 已打开
