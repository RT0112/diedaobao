# CC调用方式确认

## 时间
2026-05-22 13:33

## 背景
用户明确反对用 delegate_task/acp_command 等方式"启动新会话"来调用 Claude Code。要求直接像人一样操作：在终端里粘贴命令回车。

## 结论
调用 CC 的唯一方式：**直接在 terminal() 里跑 cc-agent.sh 命令**

```bash
# 小任务
cd ~/.qclaw/workspace-x5kuz49xple53hhg/projects && bash anthropic-qclaw-proxy/cc-agent.sh "任务描述"

# 大任务 - 交互模式（用 pty=true）
cd ~/.qclaw/workspace-x5kuz49xple53hhg/projects && bash anthropic-qclaw-proxy/cc-agent.sh --interactive

# 续接
cd ~/.qclaw/workspace-x5kuz49xple53hhg/projects && bash anthropic-qclaw-proxy/cc-agent.sh --continue "新prompt"
```

## 禁止的方式
- ❌ delegate_task(acp_command='claude')
- ❌ 启动新会话/子agent来调CC
- ❌ 任何"间接"调用方式
