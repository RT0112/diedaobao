# K70 ADB 验收 — 查看位置功能实测

## 目标
用ADB在K70上安装最新双端APK，实测"查看位置"功能是否正常

## 操作记录

### 1. 环境准备
- ADB路径: `/Users/zhou/Library/Android/sdk/platform-tools/adb`
- K70设备: `a0c2910e` ✅ 在线
- 设置 `export PATH="$HOME/Library/Android/sdk/platform-tools:$PATH"`

### 2. 安装APK
- 老人端: `fall-detection-app/app/build/outputs/apk/debug/app-arm64-v8a-debug.apk` (38MB, v0.45.6)
- 子女端: `family-guardian-app/app/build/outputs/apk/debug/app-arm64-v8a-debug.apk` (17MB, v0.8.7)
- `adb -s a0c2910e install -r` → 双端均Success ✅

### 3. 启动应用
- 老人端Activity: `.ui.MainActivity` (注意有.ui前缀!)
- 子女端Activity: `.MainActivity`
- `am start -n <包名>/<activity>` → 均启动成功 ✅

### 4. 实测"查看位置"
1. `uiautomator dump` → 找到"查看位置"按钮 bounds `[56,877][1384,1073]`
2. `adb shell input tap 720 975` → 点击成功
3. Activity切换: `MainActivity` → `MapActivity` ✅
4. 日志时序:
   - 09:27:28 getElderStatus → `lastLocation:null` (旧数据)
   - 09:27:30 getElderStatus → `lastLocation:{lat:23.087596, lng:113.387567}` ✅
   - `pullLocationStatus:"done"` ✅

### 5. 截图验证
- `adb exec-out screencap -p > /tmp/k70_screenshot.png`
- vision分析: ✅ 地图正常显示(Leaflet)、红色marker在正确位置、底部信息面板显示坐标

## 结论
✅ **CC修复的两个bug均生效，查看位置功能端到端正常工作。**
