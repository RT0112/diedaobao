2. **解决方案**：通过 ADB `input text` 命令启动 sshd → SSH 连接成功
3. **K70 后端路径**：`~/diedaobao-server/`（非之前记录的 `~/.qclaw/diedaobao-server/`）
4. **后端状态**：启动后 health 检查通过，数据库正常

## Successful Operation Chain (验证通过)
```
adb shell am start -n com.termux/.app.TermuxActivity     # 启动 Termux 前台
adb shell input text 'sshd'                               # 输入 sshd
adb shell input keyevent 66                               # 按回车
adb forward tcp:8022 tcp:8022                             # 端口转发
ssh -p 8022 -i ~/.ssh/k70_key u0_a280@localhost "cmd"    # SSH 执行命令
```
