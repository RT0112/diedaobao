# CC调用方式 - 用户可见模式验证

## 时间
2026-05-22 13:40

## 问题
用户说在Mac Terminal.app界面上看不到我跑的CC命令输入和输出，要求像他自己在终端操作一样能看到。

## 验证内容
1. ✅ cc-agent.sh 通过 terminal() 直接调用 - 已验证成功
2. ⚠️ 用户要看得到Terminal界面

## 试错过程
1. osascript -e 'do script' - 超时阻塞，❌失败
2. osascript + background=true - osascript执行期间hung住终端
3. ✅ AppleScript keystroke方案成功 — 激活Terminal窗口，用System Events把命令打进去

## 最终方案
用 keystroke 方式让CC命令出现在用户Terminal窗口：
```python
terminal(
    background=True,
    command="""osascript -e '
    tell application "Terminal"
        activate
    end tell
    delay 0.5
    tell application "System Events"
        keystroke "cd projects && bash cc-agent.sh \\"任务\\""
        key code 36
    end tell
    '"""
)
```

## 结果
✅ 用户在Terminal上能看到命令被输入并执行
