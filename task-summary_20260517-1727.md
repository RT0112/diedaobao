# Task Summary — 远程协助三Bug修复v24

## Objective
修复跌倒宝老人端远程协助功能三个Bug，编译部署到K70真机验证。

## Key Changes

### Bug ①：点击"允许"无响应
- **根因**：`handleNewRequest()` 返回 false（重复请求）时跳过 `initViews()`，按钮监听器未设置
- **修复**：`RemoteAssistActivity.kt` — `onCreate()` 和 `onNewIntent()` 中始终调 `initViews()`

### Bug ②：自动允许开关失效
- **根因**：重复请求路径直接 `showRequestDialog()`，跳过 `auto_allow_assist` 检查
- **修复**：重复请求也检查 autoAllow，为 true 时走 `onAllowClicked()`

### Bug ③：无障碍设置跳转到列表而非详情页
- **根因**：`openAccessibilitySettings()` 只返回通用列表页 Intent
- **修复**：先尝试 `ACCESSIBILITY_DETAILS_SETTINGS` 详情页 Intent，失败则 fallback 通用列表
- **关键发现**：Android 限制第三方 App 无法直接跳到无障碍详情页（需 `OPEN_ACCESSIBILITY_DETAILS_SETTINGS` 系统签名权限），fallback 是正确行为
- **隐藏常量**：`ACTION_ACCESSIBILITY_DETAILS_SETTINGS` 和 `EXTRA_ACCESSIBILITY_COMPONENT_NAME` 需用字符串硬编码
- **需额外 import**：`android.content.ComponentName`
